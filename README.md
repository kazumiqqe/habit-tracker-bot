# 🧠 Habit Tracker Bot  
Telegram бот для отслеживания привычек и поддержания дисциплины.

## Возможности  
- ➕ **Добавление привычек** — создавай привычки, которые хочешь развивать  
- ✅ **Отметка выполнения** — отмечай выполненные привычки каждый день  
- 📊 **Прогресс и серии** — визуальная статистика и подсчёт дней подряд  
- 🗑 **Удаление привычек** — избавляйся от ненужных целей  
- ⏰ **Ежедневные напоминания** — настрой время, чтобы бот напоминал о невыполненных привычках  
- ⚙️ **Гибкие настройки** — включай/выключай напоминания, выбирай удобное время  

## Стек  
- Python  
- aiogram 3.x  
- SQLite
